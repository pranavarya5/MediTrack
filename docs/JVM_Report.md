# JVM Report

## Class Loader

The class loader loads `.class` files into memory when the program runs.

## Runtime Data Areas

- Heap
- Stack
- Method Area
- PC Register

## Execution Engine

The execution engine runs bytecode using the interpreter and JIT compiler.

## JIT Compiler vs Interpreter

- Interpreter executes bytecode line by line.
- JIT compiler converts hot code into native machine code for better performance.

## Write Once, Run Anywhere

Java bytecode can run on any system that has a compatible JVM.

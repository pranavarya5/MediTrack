# Setup Instructions

## Java Setup

1. Install JDK 17 or newer.
2. Confirm installation with:

```bash
java -version
javac -version
```

3. Set `JAVA_HOME` if needed.
4. Install VS Code Java extensions if you are using VS Code.

## Screenshots to Add

- JDK installation screen
- `java -version` output
- `javac -version` output
- VS Code Java setup or terminal verification

## Project Setup

```bash
git clone <your-github-repo-url>
cd meditrack
mvn clean compile
```
